# Masterarbete2022

A small project repository containing video/image processing inputs and results for a master's thesis.

## Overview

This repository contains input videos, target images, frame extracts, and output results used for experiments and demonstrations.

## Repository structure

- frames/ — frame images extracted from videos
  - frame1.jpg

- input/ — source videos and target images used by the processing pipeline
  - kct1.mp4
  - kct2.mp4
  - kct3.mp4
  - kct4.mp4
  - kct11.mp4
  - target1.jpg
  - target2.jpg
  - target3.jpg

- output/ — results produced by the processing pipeline (currently not included in repository)
  - (results)

## Usage

Place your input video files and target images into the `input/` folder. Extracted frames should be placed into `frames/`. Run the processing scripts (not included in this repository) to generate results which will be written to `output/`.

If you need this repository to contain the missing media files, add them to the paths listed above or provide them in a separate archive.
